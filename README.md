# CS239 Social and Professional Issues: Knowing to Uphold Copyright and Ethical Guidelines 'Choose your own adventure'

I used Twine for this project.

This is a game where you make decision to look online for help with a machine learning model. You find a repo with code similar to what you are trying to recieve, and you decide to make changes to the source code. But there are consequences. Go through the game to find out what you did wrong.

In Twine I used Stylesheet to implement a pop-up box that appears when you hover over "all rights reserved" which gives you the definition.
